# GitHub Copilot License Tiers

GitHub Copilot offers several different license tiers to accommodate various user needs, from individual developers to large organizations. This document outlines the available options to help you choose the right tier for your needs.

## Available Tiers

| Tier           | Price              | Intended For                                          |
| -------------- | ------------------ | ----------------------------------------------------- |
| **Free**       | $0                 | Developers looking to get started with GitHub Copilot |
| **Pro**        | $10 per month      | Individual developers who want unlimited access       |
| **Business**   | $19 per user/month | Teams ready to accelerate their workflows             |
| **Enterprise** | $39 per user/month | Organizations seeking a fully customized experience   |

## Free Tier

The GitHub Copilot Free tier is perfect for getting started with AI-assisted coding:

- **Limitations**:

  - Up to 2,000 code completions per month
  - Up to 50 chat messages per month

- **Key Features**:
  - Access to basic AI models (including OpenAI GPT-4o, Claude 3.5 Sonnet, OpenAI o3-mini, Google Gemini 2.0 Flash)
  - Real-time code suggestions
  - Inline chat and basic debugging assistance
  - Context-aware coding support
  - Multi-file editing in VS Code
  - Public code filter with code referencing

## Pro Tier

For $10/month, the Pro tier removes limitations and adds additional features:

- **Key Additions Beyond Free**:
  - **Unlimited** code completions and chat messages
  - Access to additional AI models (including Claude 3.7 Sonnet, OpenAI o1)
  - Pull request summaries
  - Free for verified students, teachers, and maintainers of popular open source projects

## Business Tier

At $19/user/month, the Business tier adds team management capabilities:

- **Key Additions Beyond Pro**:
  - User management
  - Data excluded from training by default
  - Enterprise-grade security
  - IP indemnity
  - Content exclusions
  - SAML SSO authentication
  - Usage metrics

## Enterprise Tier

For $39/user/month, Enterprise tier provides the most comprehensive offering:

- **Key Additions Beyond Business**:
  - Preview access to OpenAI GPT-4.5
  - Attach knowledge bases to chat for organizational context
  - Fine-tuned models for code completion (coming soon)
  - Set coding guidelines for code review
  - Requires GitHub Enterprise Cloud

## Special Eligibility Programs

GitHub Copilot is available for free to certain users:

- **Students and Educators**: Verified students and teachers can access Copilot Pro for free through GitHub Education
- **Open Source Contributors**: Maintainers of popular open source projects can qualify for free access to Copilot Pro
- **Startups**: Eligible startups may qualify for discounts through the GitHub for Startups program

## Supported Environments

All GitHub Copilot tiers support the following environments:

- **Editors and IDEs**: VS Code, Visual Studio, JetBrains IDEs, Neovim, Xcode, Azure Data Studio
- **GitHub Platforms**: github.com, GitHub Mobile, GitHub CLI
- **Terminal**: Windows Terminal

## How to Choose

Consider these factors when selecting a tier:

- **Volume of Use**: If you code regularly, the Free tier's limits may be restrictive
- **Team Coordination**: Business and Enterprise tiers provide team management features
- **Security Concerns**: Business and Enterprise tiers offer enhanced security and privacy options
- **Advanced Features**: Enterprise tier provides the most customization and integration options

For the most up-to-date pricing and feature information, visit the [official GitHub Copilot pricing page](https://github.com/features/copilot#pricing).

## Getting Started

1. To sign up for GitHub Copilot, visit [github.com/features/copilot](https://github.com/features/copilot)
2. Choose your preferred tier
3. Follow the setup instructions to install GitHub Copilot in your preferred environment
4. For team deployments, an organization admin will need to set up and manage licenses
