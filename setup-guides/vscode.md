# Setting Up GitHub Copilot in VS Code

This guide provides basic steps to get GitHub Copilot up and running in Visual Studio Code with your GitHub Copilot Business license.

## Prerequisites

- [Visual Studio Code](https://code.visualstudio.com/) installed on your machine
- Your GitHub account with access to GitHub Copilot Business
- Organization membership with Copilot Business enabled

## Installation Steps

### Automated Setup (Recommended)

1. Open VS Code
2. Click on the Copilot icon in the title bar, or open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
3. Select **GitHub Copilot: Sign In**
4. Complete the authentication process in your browser
5. If prompted, select the GitHub account associated with your Business license

![Copilot menu in VS Code title bar](https://code.visualstudio.com/assets/docs/copilot/setup/vs-code-copilot-menu.png)

### Manual Setup (Alternative)

1. Open VS Code
2. Navigate to the Extensions view (`Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Search for "GitHub Copilot" and install both:
   - GitHub Copilot
   - GitHub Copilot Chat (installed automatically with Copilot)
4. Open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
5. Enter **GitHub Copilot: Sign in**
6. Follow the authentication steps in your browser

## Verifying the Installation

After installation, you can verify Copilot Business is working by:

1. Opening the Chat view with `Ctrl+Alt+I` / `Ctrl+Cmd+I`
2. Checking that you can enter prompts in the chat input box
3. Confirming you have access to all available models in the dropdown menu
4. Opening a code file and typing a comment to see if code suggestions appear
5. Checking that the Copilot status in the bottom bar shows "Ready"

## Switching GitHub Accounts

If you need to use a different GitHub account with Copilot:

1. Select the Accounts menu in the Activity Bar
2. Sign out of your current GitHub account
3. Sign in with your alternative GitHub account that has Copilot access

## Useful Keyboard Shortcuts

- Open Copilot Chat: `Ctrl+Alt+I` / `Ctrl+Cmd+I`
- Accept code suggestion: `Tab`
- Dismiss code suggestion: `Esc`
- Show next suggestion: `Alt+]` / `Option+]`
- Show previous suggestion: `Alt+[` / `Option+[`
