# GitHub Copilot Resources

This directory provides supplementary resources and reference materials to help you get the most out of GitHub Copilot. These resources are designed to complement the exercises and serve as ongoing references after the workshop.

## Resource Summary

| Resource                                        | Description                                                           | When to Use                                                                    |
| ----------------------------------------------- | --------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| [Cheatsheet](./cheatsheet.md)                   | Quick reference for keyboard shortcuts, commands, and common patterns | When you need to quickly look up how to perform a specific action with Copilot |
| [Available Models](./available-models.md)       | Overview of Copilot's AI models and their capabilities                | When selecting models for different tasks                                      |
| [Effective Prompts](./effective-prompts.md)     | Guide with examples of effective prompting techniques                 | When crafting prompts for complex scenarios or troubleshooting suggestions     |
| [Video References](./video-references.md)       | Curated collection of official tutorial videos                        | For visual learning or deeper understanding of specific features               |
| [Real-World Examples](./real-world-examples.md) | Practical examples from actual development projects                   | To see how Copilot is applied in professional contexts                         |

## Using These Resources

### During the Workshop

- Keep the **Cheatsheet** open as a quick reference for keyboard shortcuts and commands
- Refer to **Effective Prompts** when working on the exercise challenges
- Browse **Real-World Examples** during breaks to see practical applications

### After the Workshop

- Use the **Cheatsheet** as a daily reference when working with Copilot
- Review **Effective Prompts** when tackling new types of development tasks
- Watch **Video References** to deepen your understanding of advanced features
- Study **Real-World Examples** for inspiration on applying Copilot to your projects

## Resource Combinations

Different resources work well together for specific scenarios:

1. **Learning a new Copilot feature**:

   - Start with relevant videos from Video References
   - Use the Cheatsheet to learn the commands
   - Refer to Effective Prompts for usage techniques

2. **Optimizing your prompting skills**:

   - Study the patterns in Effective Prompts
   - Compare with Real-World Examples
   - Practice using the techniques in your own work

3. **Troubleshooting issues**:
   - Check the Cheatsheet for correct command usage
   - Review Effective Prompts for better ways to communicate your intent
   - See if Video References covers similar scenarios

## Contributing to Resources

These resources are living documents that benefit from community contributions. If you discover effective prompting techniques, useful shortcuts, or have real-world examples to share, consider submitting a pull request to enhance these materials for future workshop participants.

Each resource file contains more detailed information about its specific topic. Click on the links above to explore each resource in depth.
