# GitHub Copilot Official Workshops

## Overview

GitHub provides several official workshop repositories designed to help developers learn and practice using GitHub Copilot in different programming contexts. These workshops are hands-on, guided experiences that demonstrate how Copilot can accelerate development across various languages and platforms.

## Available Workshops

GitHub currently offers three main workshop repositories that you can clone and work through:

### 1. Node Calculator Workshop

**Description**: Add features and unit tests to a Node.js calculator application using GitHub Copilot.

**Skills practiced**:
- JavaScript/Node.js development with Copilot
- Adding new functionality to existing code
- Writing unit tests with Copilot's assistance
- API development

**Link**: [Node Calculator Workshop](https://github.com/copilot-workshops/copilot-node-calculator)

### 2. Rock Paper Scissors Workshop

**Description**: Build a Rock Paper Scissors game from scratch using Python and GitHub Copilot.

**Skills practiced**:
- Python development with Copilot
- Game logic implementation
- Command-line interface design
- Extending functionality (adding Lizard and Spock options)

**Link**: [Rock Paper Scissors Workshop](https://github.com/copilot-workshops/copilot-rock-paper-scissors)

### 3. Terraform Workshop

**Description**: Learn how to work with Infrastructure as Code (IaC) using Terraform and GitHub Copilot.

**Skills practiced**:
- Infrastructure as Code development
- Terraform configuration creation
- Cloud resource provisioning
- Best practices for IaC

**Link**: [Terraform Workshop](https://github.com/copilot-workshops/copilot-terraform)

## Getting Started

To participate in any of these workshops:

1. Visit the GitHub repository link for your chosen workshop
2. Follow the "Use this template" option or clone the repository
3. Set up GitHub Copilot in your preferred development environment
4. Follow the instructions provided in the repository's `.instructions` directory

Each workshop includes detailed setup guides and step-by-step exercises to help you learn Copilot's features in a practical, hands-on way.

## Workshop Format

The workshops typically follow this structure:
- Setup instructions for your environment
- Core exercises to learn basic concepts
- Challenge exercises to extend your skills
- Additional resources for further learning

Workshops can be completed individually at your own pace or as part of guided group sessions.
