# GitHub Copilot Exercises

This directory contains hands-on exercises to help you master GitHub Copilot's features. Each exercise is designed to build practical skills that you can immediately apply to your development workflow.

## Recommended Exercise Order

For a progressive learning experience, we recommend completing the exercises in this order:

1. **Code Completions** - Start with the fundamentals of getting and refining suggestions
2. **Copilot Chat** - Learn how to have conversations about your code
3. **Multi-File Editing** - Expand to working across multiple files
4. **GitHub Web Integration** - See how Copilot works directly on GitHub.com
5. **Agent Mode** (Business tier only) - Explore advanced contextual conversations
6. **Dependabot Exercise** - Apply Copilot to DevOps workflows

## Exercise Structure

Each exercise follows a consistent format:

- **Overview** - Brief introduction to the feature
- **Learning Objectives** - What you'll learn from the exercise
- **Prerequisites** - What you need before starting
- **Exercise Steps** - Step-by-step instructions with examples
- **Challenge Activities** - Additional tasks to reinforce learning
- **Tips for Success** - Helpful guidance for getting the most out of the feature
- **Next Steps** - Suggestions for further exploration

## For Different Roles

These exercises are designed to be valuable for various roles:

- **Frontend Developers**: Focus on completions, chat, and multi-file editing for component development
- **Backend Developers**: Explore chat for API design and completions for server-side logic
- **Android Developers**: Use completions and chat to accelerate mobile development
- **Designers**: Try GitHub web integration for collaboration and chat for implementation questions
- **Testers/QA**: Leverage chat and completions to generate test cases and validation logic

## Tips for Success

- Complete the exercises on your own pace - it's not a race
- Try the challenge activities to deepen your understanding
- If you get stuck, refer to the [Effective Prompts](../resources/effective-prompts.md) resource
- Use the [Cheatsheet](../resources/cheatsheet.md) as a quick reference during exercises
- Apply what you learn to your own code samples for maximum relevance

## Workshop Customization

Workshop facilitators can customize these exercises by:

- Focusing on specific exercises relevant to their team's tech stack
- Supplementing with examples from their own codebase
- Adjusting the difficulty of challenge activities
- Extending time on exercises that align with team priorities
